****important points****

  • percentage is a relative unit, it will look for parents absolute unit to calculate the current elemnets
  • we can give % in padding, it will calculate according to its parents width
  1. % in width ---> parent width
  2. % in height ---> parent height
  3. % in padding and margin ----> parent width
  4. % in font-size ----> parent font-size
  1. vw is a width of browser not body or html
