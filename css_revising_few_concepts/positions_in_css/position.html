postions in css

position is a property in css which is used to set the position of an element in a webpage. There are 5 types of positions in css.

  1. static
  2. relative
  3. absolute
  4. fixed
  5. sticky

1.static is the default position of an element in a webpage. It is not affected by top, bottom, left, right properties.

2.relative is the position of an element relative to its normal position. It is affected by top, bottom, left, right properties.

3.absolute is the position of an element relative to its parent element. It is affected by top, bottom, left, right properties.

4.fixed is the position of an element relative to the viewport. It is affected by top, bottom, left, right properties.

5.sticky is the position of an element relative to its parent element. It is affected by top, bottom, left, right properties. ***sticky only moves in parents container in viewport if scrollbar .if container is outside of viewport then it will look for scroll bar in parents if not available then moves according to viewport***

when we use position property in css then we have to use top, bottom, left, right properties to set the position of an element.

inset is used to give top,right,left,bottom value together
